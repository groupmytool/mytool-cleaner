# Mytool Cleaner
Mytool Cleaner is a software uninstallation auxiliary tool, which includes the following features:
* Uninstall applications
* Analyze disk space

# Run

## Prerequisites
* OpenJDK17+
* Maven3.8.x

## Running
Main class:
```
com.mytool.cleaner.CleanerApp.java
```

## Build
```shell
./mvnw clean package
```
output: `target/mytool-cleaner.app`
